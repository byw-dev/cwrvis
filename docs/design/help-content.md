## 摘要

大气水物质由水汽和大气水凝物（云水）组成，水汽通过热力动力过程凝结形成大气水凝物，大气水凝物通过云物理过程形成降水，降水主要来自大气水凝物，水汽-水凝物-降水的相互转化构成复杂的大气水物质循环过程。为满足人工影响天气对于开发对象资源了解的需要，提出云水资源和大气水资源的概念。定义一定区域、一定时段内，参与大气水循环全部过程，但未形成地面降水、仍留存在空中的大气水凝物总量（或大气水物质总量），称为云水资源（或大气水资源（Zhou et al.,2020）。为完整给出大气水物质循环过程各物理量定量值，进一步完善了大气水物质循环各物理量概念及计算式。包括：状态量是指某时刻可以观测到的瞬时量，平流量指一定时段内沿区域边界流入/流出的物理量；源汇量指一定时段内水汽和水凝物发生相态变化产生的地表蒸发、地面降水、云凝结和云蒸发量。按照质量守恒，在大气水物质循环过程中，对于一定区域和时段，输入项（包括：状态初值、输入和源）之和等于支出项（包括：状态终值、输出和汇）之和，定义为一定区域时段参与大气水循环全部过程的总量。降水与总量的比值定义为降水效率，包括：水汽降水效率、水凝物降水效率和总水物质降水效率，云凝结量与水汽总量的比例定义为水汽凝结效率。一定区域时段内的水物质平均状态量与降水强度的比值定义为更新期，包括：水汽更新期和水凝物更新期。

## 数据来源说明

基于大气水物质循环完整过程和云水资源概念及定量评估理论，在以往主要关注水汽和降水的常规大气水循环基础上，增加云水循环过程，形成完整的大气水物质循环的评估。采用CLOUDSAT/CALIPSO卫星云观测、NCEP/NCAR大气再分析资料及全球降水气候数据集(GPCP)降水产品，结合飞机云物理探测，通过云水资源观测诊断评估系统 (CWR-DQ)，构建了不同分辨不同时段的大气水物质循环特征量及云水资源评估数据集。本平台使用数据源为2000–2025年云水资源观测诊断评估数据集，以下为平台中变量的物理意义说明表。

## 表1 变量物理意义说明表

原文"符号"列为 MathType WMF 图片（不兼容浏览器），实现时改用 HTML `<i>` + `<sub>` 表示。

| 物理量名称 | 符号（HTML 写法） | 物理意义 |
|---|---|---|
| 水汽平均状态量 | `MM<sub>v</sub>` | T时段内，研究区域中的水汽状态量的平均值，单位：kg |
| 水凝物平均状态量 | `MM<sub>h</sub>` | T时段内，研究区域中的水凝物状态量的平均值，单位：kg |
| 水汽输入值 | `Q<sub>vi</sub>` | T时段内，研究区域边界上水平流入的水汽总量，单位：kg |
| 水汽输出值 | `Q<sub>vo</sub>` | T时段内，研究区域边界上水平流出的水汽总量，单位：kg |
| 水凝物输入值 | `Q<sub>hi</sub>` | T时段内，研究区域边界上水平流入的水凝物总量，单位：kg |
| 水凝物输出值 | `Q<sub>ho</sub>` | T时段内，研究区域边界上水平流出的水凝物总量，单位：kg |
| 云凝结 | `C<sub>vh</sub>` | T时段内，研究区域中，水汽通过凝结（含凝华）过程发生相态转化成为水凝物的质量，单位为：kg |
| 地面降水 | `P<sub>s</sub>` | T时段内，研究区域中的累计降水量，单位：kg |
| 水汽总量 | `GM<sub>v</sub>` | T时段内，参与研究区域大气水循环过程的所有水汽收入项，单位：kg |
| 水凝物总量 | `GM<sub>h</sub>` | T时段内，参与研究区域大气水循环过程的所有水凝物收入项，单位：kg |
| 水凝物降水效率 | `PE<sub>h</sub>` | 降水量与水凝物总量的比值，单位：% |
| 水汽凝结效率 | `CE<sub>v</sub>` | 水汽发生相态转化形成大气水凝物的效率，单位：% |
| 水汽更新期 | `RT<sub>v</sub>` | T时段内，研究区域中水汽状态量的平均值和平均降水强度的比值，反映了一定时段内大气水汽循环情况。单位：hour |
| 水凝物更新期 | `RT<sub>h</sub>` | T时段内，研究区域中水凝物状态量的平均值和平均降水强度的比值，反映了一定时段内大气水凝物循环情况。单位：day |
| 云水资源量 | `CWR` | T时段内，参与研究区域大气水物质循环过程，仍留存在空中的大气水凝物总量，单位：kg |
